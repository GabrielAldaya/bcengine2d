TaT Tile Map Editor 1.0.2

TaT Tile Map Editor requires Java 1.5.0 (you should use 1.5.0_04 or later).

To run TaT Tile Map Editor on Windows, use the editor.exe.
To run TaT Tile Map Editor on Linux, go to the TaT Tile Map Editor directory
and write "java -Xmx128M -jar TileMapEditor.jar" to the terminal window.

The -Xmx128M argument tells the Java to use 128MB heap size. Use larger value
if your tile map projects need more memory. The same argument can be used with
editor.exe on Windows.
